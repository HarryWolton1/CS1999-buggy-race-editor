What is my buggy editor

My buggy race editor is a tool that will allow you to make buggies which you can enter into races on the buggy race server.
This buggy editor is designed to be user friendly and inclusive by using fonts that are easy to read for everyone and the website is not too complex which may confuse users.

Features

  • Creating buggies:
    • You can create multiple buggies
    • Buggies cannot be created that break the rules
  • Editing buggies
    • Any buggies you create can be edited
    • Buggy rules are enforced when editing your buggies.
  • Displaying buggies
    • All the buggies you've created can be displayed on tables
    • Your buggies flag will be drawn for you
  • Displaying specifications
    • There is a page to display all the options you can choose from the buggy race server
    • This uses a system which means the tables on this editor will change when the buggy race server changes
